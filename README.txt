How to run and test the program:

1) First navigate to the folder where the python files are located from command prompt.

2) then you can run the program by:
    python change_directory *current path* *new path* and you will get the result printed on console.
  
    for ex: 
    python change_directory.py /abc/def ghi

    output:
    /abc/def/ghi

3) You can run all the automated unit tests by using:
     pytest test_change_directory.py
